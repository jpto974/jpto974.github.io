Bienvenu sur AVL - Une application vous permettant de réserver votre vélo urbain. 1 - Choisissez votre ville! Une fois la carte affichée, choisissez votre station. 2 - Le formulaire apparaît, renseignez votre nom et votre prénom puis réservez. 3 - Vous avez 20 minutes pour récupérer votre vélo. Sinon la réservation sera annulée. 4 - Si vous refaites une autre réservation la précédente sera annulée. La fermeture de l'appli annule aussi la réservation.
